# 3.2 表ADT
对于表list A_0, A_1, ..., A_N-1，表的大小为N。  
定义元素 A_i 在表中的位置（position）为i。  
讨论中将假设表中元素为整数。  

对表ADT的操作：  
printList, makeEmpty;  
find返回某元素首次出现的位置;  
insert, remove;  
findKth返回指定位置上的元素.
## 3.2.1 表的简单数组实现  
适用情形：对表的尾端（high end of the list）进行操作，此后只发生对表的访问操作（findKth）。  
## 3.2.2 简单链表
linked list  
每个节点含有表元素和一个链（link），该链称为next链（next link）。
